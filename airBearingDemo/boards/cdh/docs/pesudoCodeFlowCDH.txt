[Early Boot]
  • clocks → pins → watchdog
  • SysTick(1 ms) → start
  • init NVM & boot markers
  • FDCAN init + filters
  • Radio/UART-DMA init
  • Storage (FRAM/MRAM) init
  • Read active bank + TMR health
  • Load last known SAFE cause
   ↓
[Enter BOOT state]
   ↓
[Self-check & Handshakes]
  • ping EPS for Vbatt/OK-to-power
  • ping COMM for radio ready
  • ping ADCS (optional)
  → if any fail ⇒ SAFE
   ↓
[Transition to SAFE]
  • constrain loads; log boot SOH
  • start house tasks (beacon, health, scrub)
  • wait for ground or autonomous OK
   ↓
