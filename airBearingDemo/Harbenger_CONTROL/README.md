# CubeSat Control Project

## Overview
This repository contains all files related to the CubeSat control project, including simulation code, test data, progress reports, research, and hardware specifications.

## Folder Structure

### Detumbling
Contains code found online for detumbling procedures. Note that this code requires MATLAB Aerospace Toolbox to run, which was not available during development.

### Machining
Contains technical files and specifications for machining the pyramid reaction wheel frame.

### Matlab Scripts
Contains various MATLAB scripts developed for the project:
- Star tracker calibration routines
- Coordinate frame conversion utilities
- Other supporting algorithms

### Progress Reports - Fall
Contains all progress reports submitted during Fall 2024 semester.

### Progress Reports - Spring
Contains all progress reports submitted during Spring 2025 semester.

### Research
Contains:
- Mathematical calculations and analysis
- Documentation on the spherical air bearing system
- Other research materials supporting the project

### Reviews
Contains:
- Design review presentations
- Test readiness documentation
- Other project review materials

### RustControl
Contains Rust code converted from MATLAB using AI assistance. This is experimental and may require further verification and testing.

### Simulation
Contains working MATLAB code that simulates the control systems for the CubeSat. The structure is similar to the organization used in Dr. Eves' class.

### Simulation - Test
Contains a copy of the simulation with attempts to convert it to a single MATLAB file (Claudetest) to facilitate conversion to Rust.

### Testing
Contains:
- Test planning documentation
- Recommendations from Dr. Eves regarding necessary pre-flight tests
- Documentation for the spherical air bearing testing apparatus

## Getting Started
To work with the simulation, begin with the files in the Simulation folder as they contain the most stable and functional code.

## Notes
- The detumbling code requires MATLAB Aerospace Toolbox to run
- RustControl code is experimental and may need additional work
- Testing procedures should follow Dr. Eves' recommendations found in the Testing folder
